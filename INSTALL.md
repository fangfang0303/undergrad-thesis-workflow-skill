# Install Guide

## Windows

1. Download or clone this repository.
2. Copy the `undergrad-thesis-workflow` folder into:

```text
C:\Users\<username>\.codex\skills\
```

3. Restart Codex or open a new thread.

## Verify

Ask Codex:

```text
请帮我使用 undergrad-thesis-workflow 这个 skill，按本科论文问卷研究流程推进。
```

If the skill is installed correctly, Codex should use the skill workflow for thesis project guidance.
