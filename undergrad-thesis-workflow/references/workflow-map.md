# Workflow Map

This skill was packaged from a real undergraduate thesis workflow with these major steps:

1. initialize project structure
2. capture advisor constraints
3. narrow the topic to a specific population
4. write proposal / opening report
5. build questionnaire and coding rules
6. collect questionnaire data
7. remove test rows and clean data
8. calculate BMI and main behavior scores
9. build result tables and charts
10. write discussion and conclusion
11. revise after plagiarism and AIGC checks

## Good Trigger Examples

- "Help me initialize my undergraduate thesis project"
- "My advisor says the topic is too broad"
- "Help me turn this topic into a questionnaire study"
- "I exported questionnaire data, help me clean it"
- "Help me write the results section from this survey"
- "My paper needs charts"
- "My plagiarism report is back, what do I revise"
- "My advisor says it sounds like AI"

## Bad Trigger Cases

This skill is not ideal when:

1. the user only wants a one-paragraph answer unrelated to thesis workflow
2. the project is a pure lab experiment with no overlap in process
3. the user only needs citation formatting with no workflow support
