# Share Guide

## Manual Install

Copy the folder `undergrad-thesis-workflow` into:

`%USERPROFILE%\\.codex\\skills\\`

On this machine that path is typically:

`C:\\Users\\surface\\.codex\\skills\\`

After copying, restart the Codex app or open a new thread so the skill can be discovered.

## What To Tell Another User

Recommended wording:

`请帮我使用 undergrad-thesis-workflow 这个 skill，按本科论文问卷研究流程推进。`

Or:

`请帮我用 undergrad-thesis-workflow skill，从选题收窄、问卷设计、数据清洗到论文写作完整推进。`

## Optional Add-On Materials

If sharing with another student, also include:

1. advisor summary
2. questionnaire example
3. coding table example
4. final checklist

These are not required for installation, but they make the workflow easier to reuse.
